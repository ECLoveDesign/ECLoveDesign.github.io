網站安全政策宣告

:::

親愛的訪客，感謝您蒞臨「國軍臺中總醫院 藥典查詢系統」網站（以下簡稱本網站）。

  • 一、嚴禁公布機密性資料：

      依據行政院發布之「國家機密保護法施行細則」及國防部發布之「軍事機密與國防秘密種類範圍等級劃分準則」，對於機密性資訊嚴禁公佈於本網站。 本網站任何資訊之公告、內容變更都需經國防部軍醫局之保防單位審核後方可上網公布。

  • 二、網站安全防護：

    • 任何未經本網站授權而企圖上傳或變更本網站所提供的各項服務及相關網頁資訊的行為，都是嚴厲禁止而且該行為可能觸犯法律。
    • 架設網路防火牆及網路入侵偵測系統防止非法入侵及監控網路流量，並追蹤未經授權而企圖上傳或變更本網站所提供的各項服務及相關網頁資訊之蓄意破壞者的上網行為及相關資訊。
    • 定期進行網站資料備份作業。
    • 隨時安裝最新作業系統漏洞修補程式。
    • 安裝防毒軟體，定期掃毒，以提供使用者安全的網頁瀏覽環境。

系統說明

本院藥品查詢系統，為提供醫療人員(包括醫師、藥師、護理人員)獲得有關本院常備用藥品項、用藥規定等說明，以利醫師能正確選擇適當用藥，作安全有效之治療。

本資料庫內容收錄包刮中、英文商品、成分名、衛生署核准適應症、健保藥理分類、常見副作用、禁忌(症)、民眾用藥指引、健保給付規定限制、注意事項、藥品外觀辨識、實體外觀圖片及電子仿單等資料，提供更為詳盡的醫藥查詢服務。

本資料庫藥品資料僅提供藥品查詢服務，所載資料並無法當作醫療建議，如有對本資料庫內容疑義或錯誤，請聯絡我們提出指正或就近向醫療專業人員尋求協助。
